# Minor Project 04

## Project Name
Todo App and Notes App

## Features

### Todo App
- Add Task
- Edit Task
- Delete Task
- Mark Task as Completed
- Show Task Status (Completed/Pending)

### Notes App
- Add Notes
- Edit Notes
- Delete Notes
- Organized Notes Layout

## Technologies Used
- React
- Vite
- CSS

## How to Run

1. Open terminal
2. Type:

npm install

3. Then type:

npm run dev

4. Open the localhost link in browser
